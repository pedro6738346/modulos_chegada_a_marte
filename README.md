# Missão Aurora Siger — MGPEB
## Módulo de Gerenciamento de Pouso e Estabilização de Base

Este projeto foi desenvolvido como atividade integradora da Fase 2, simulando o sistema responsável por gerenciar pousos de módulos da colônia Aurora Siger em Marte.

## Funcionalidades

- Cadastro de módulos em estruturas lineares (fila, lista e pilha)
- Busca de módulos por combustível, prioridade e criticidade
- Ordenação da fila de pouso por prioridade (Bubble Sort)
- Simulação de autorização de pouso com base em regras lógicas booleanas

## Estruturas de dados utilizadas

- **Fila** — módulos aguardando autorização de pouso (FIFO)
- **Lista** — módulos já pousados
- **Pilha** — módulos em situação de alerta (LIFO)
